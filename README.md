**RainCord — платформа для комунікації в реальному часі (чат, аудіо та відеозв’язок)**  
Створена з використанням сучасного стеку веб-технологій для швидкої та стабільної взаємодії між користувачами.  
Проєкт реалізує систему серверів, каналів та повідомлень у реальному часі.
